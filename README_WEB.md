## 🚀 快速开始

### 1. 安装依赖

```bash

```

### 2. 启动应用

```bash

```

### 3. 访问应用

浏览器自动打开 `http://localhost:8501`

## 🔧 自定义配置

### 修改主题颜色

编辑 `.streamlit/config.toml`:

```toml
[theme]
primaryColor = "#your_color"  # 主色调
backgroundColor = "#ffffff"    # 背景色
```

### 修改端口

```bash
streamlit run app.py --server.port 8080
```

### 添加新页面

1. 在 `pages/` 目录创建新文件
2. 实现 `show()` 函数
3. 在 `app.py` 中添加导航项

## 🐛 常见问题

### Q: 启动失败？

**A**: 检查依赖是否安装完整:
```bash
pip install -r requirements.txt --upgrade
```

### Q: 图表不显示？

**A**: 确保 Plotly 版本正确:
```bash
pip install plotly>=5.14.0
```

### Q: 页面加载慢？

**A**:
- 清除缓存: 点击右上角 "Clear cache"
- 减少数据量
- 使用本地数据

### Q: 如何部署到云端？

**A**: 可以部署到:
- **Streamlit Cloud** - 免费托管
- **Heroku** - 容器化部署
- **AWS/阿里云** - 服务器部署
