# 盈米金融教学平台 V2.0 - 快速开始

## 🎯 5分钟快速体验

### 步骤1: 安装依赖 (1分钟)

```bash
cd /Users/ethen/Documents/MAC/金融教学应用/web_app
pip install -r requirements.txt
```

### 步骤2: 配置密钥 (1分钟)

```bash
# 复制示例文件
cp .streamlit/secrets.toml.example .streamlit/secrets.toml

# 使用默认密钥（可选：编辑自己的密钥）
# 文件已包含默认密钥，可直接使用
```

### 步骤3: 启动应用 (1分钟)

```bash
# 方法1: 使用启动脚本（推荐）
bash run.sh

# 方法2: 直接启动
streamlit run app.py
```

### 步骤4: 登录体验 (2分钟)

浏览器自动打开 `http://localhost:8501`

**使用演示账户快速登录**:
1. 点击"演示账户登录"按钮
2. 自动使用 demo_student / demo123 登录
3. 开始探索各项功能！

或者**注册新账户**:
1. 切换到"注册"标签
2. 填写用户信息（至少填写带*的必填项）
3. 点击"注册"按钮
4. 返回"登录"标签登录

## 🐛 遇到问题？

### 常见问题快速解决

#### Q1: 启动失败？

```bash
# 检查Python版本（需要3.8+）
python --version

# 重新安装依赖
pip install -r requirements.txt --upgrade
```

#### Q2: 无法登录？

```
1. 使用演示账户: demo_student / demo123
2. 检查是否正确注册
3. 确认密码符合要求（至少6位，包含字母和数字）
```

#### Q3: API调用失败？

```
1. 检查 .streamlit/secrets.toml 中的 MCP_API_KEY
2. 确认网络连接正常
3. 降级到模拟数据（自动）
```

#### Q4: Docker启动失败？

```bash
# 检查Docker是否运行
docker ps

# 查看日志
docker-compose logs

# 重新构建
docker-compose up -d --build
```
